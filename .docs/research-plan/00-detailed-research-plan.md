# PI MONO 详细研究计划

## 一、研究任务拆解

### 阶段一：架构分析（2天）

#### 任务 1.1：项目结构与依赖分析

**任务描述**：分析 PI MONO 项目的整体架构和包依赖关系

**具体工作**：
- 分析七个包（pi-ai, pi-agent-core, pi-coding-agent, pi-tui, pi-mom, pi-web-ui, pi-pods）的职责边界
- 梳理包之间的依赖关系和数据流动
- 识别核心入口文件和导出结构
- 绘制架构依赖图

**交付**：
- 架构依赖关系文档（01-architecture-overview.md）

**验收要求**：
- 包含完整的包依赖关系图
- 说明各包的职责和位置
- 涵盖核心数据流和四种工作模式

#### 任务 1.2：核心数据流设计分析

**任务描述**：分析从用户输入到 LLM 调用的完整链路

**具体工作**：
- 追踪交互模式下的完整请求-响应流程
- 分析消息的序列化与反序列化过程
- 研究事件流的设计和传播机制

**交付**：合并至架构文档

**验收要求**：
- 包含完整的数据流图示或描述
- 说明关键组件的交互顺序

### 阶段二：专题分析（4天）

#### 任务 2.1：pi-ai 类型系统与 LLM 提供商分析

**任务描述**：深入分析 pi-ai 包的类型系统和 20+ 提供商抽象

**具体工作**：
- 分析消息协议的类型定义
- 研究事件流的类型设计
- 对比不同提供商的 API 差异处理方式
- 分析工具调用、thinking/reasoning 的类型实现
- 研究模型注册和发现机制

**交付**：
- pi-ai 类型系统与 API 提供商分析文档（02-pi-ai-typesystem.md）

**验收要求**：
- 说明核心类型和数据结构
- 列出支持的 LLM 提供商及其特点
- 解释提供商抽象的实现方式
- 包含代码示例

#### 任务 2.2：pi-agent-core 运行时分析

**任务描述**：分析 Agent 运行时的状态管理和核心功能

**具体工作**：
- 分析状态管理机制
- 研究工具执行流程
- 分析事件流处理
- 研究配置管理和可扩展性

**交付**：
- pi-agent-core 分析文档（03-agent-core.md）

**验收要求**：
- 说明 Agent 运行时的核心组件

#### 任务 2.3：coding-agent CLI 分析

**任务描述**：分析 CLI 应用的四种模式和用户交互

**具体工作**：
- 分析交互模式、打印模式、RPC 模式、JSON 模式
- 研究工具系统的实现
- 分析会话管理机制
- 研究用户交互流程

**交付**：
- coding-agent CLI 分析文档（04-coding-agent-cli.md）

**验收要求**：
- 说明四种模式的区别和应用场景
- 解释工具系统的注册和使用
- 包含会话管理的实现细节

#### 任务 2.4：pi-tui 终端 UI 系统分析

**任务描述**：分析终端 UI 的渲染策略和组件系统

**具体工作**：
- 研究差分渲染策略
- 分析内置组件（Text, Box, Input, Editor 等）
- 研究主题系统实现

**交付**：

**验收要求**：
- 说明差分渲染的三层策略

#### 任务 2.5：扩展与技能系统分析

**任务描述**：分析 Extensions 和 Skills 机制

**具体工作**：
- 研究 Extensions 的加载和初始化
- 分析工具和命令的注册接口
- 研究事件系统设计
- 分析 Skills 编写规范
- 研究 Pi Packages 分发机制

**交付**：

## 三、验收标准

### 文档验收通用要求

1. 使用简洁的技术风格，避免冗余描述
2. 包含必要的代码示例和图示
3. 说明实现原理，不仅仅是 API 使用方式
4. 列出参考的源代码位置
5. 格式规范，符合 Markdown 标准

### 各文档特定验收要求

| 文档 | 核心验收点 |
|------|-----------|
| 01-architecture-overview.md | 完整的包依赖关系图、四种工作模式的区别 |
| 02-pi-ai-typesystem.md | 20+ 提供商的抽象方式、类型系统设计 |
| 03-agent-core.md | 状态管理机制、工具执行流程 |
| 04-coding-agent-cli.md | 四种模式的实际使用、会话管理细节 |
| 05-tui-system.md | 差分渲染策略、组件生命周期 |
| 06-extensions-skills.md | Extensions 加载流程、Skills 编写规范 |
| 07-technical-summary.md | 设计理念总结、可借鉴的最佳实践 |
