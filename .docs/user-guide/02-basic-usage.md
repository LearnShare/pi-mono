# 基础使用

> 基于 pi.dev/docs/latest 官方文档整理

---

## 1. 交互模式

### 启动

```bash
cd /path/to/project
pi
```

### 界面区域

界面四个区域：
- **启动头部** - 快捷键、上下文文件、模板、技能、扩展
- **消息区域** - 对话历史、工具调用、扩展 UI
- **编辑器** - 输入区域，边框颜色表示思考级别
- **底部栏** - 工作目录、会话名、token/缓存用量、成本、模型

### 编辑器功能

| 功能 | 操作 |
|------|------|
| 文件引用 | 输入 `@` 模糊搜索项目文件 |
| 路径补全 | Tab 键 |
| 多行输入 | Shift+Enter |
| 图片粘贴 | Ctrl+V（Windows 用 Alt+V） |
| Shell 命令 | `!command` 执行并发送输出给模型 |
| 隐藏命令 | `!!command` 执行但不发送输出 |
| 外部编辑器 | Ctrl+G 打开 `$VISUAL` 或 `$EDITOR` |

## 2. 非交互模式

```bash
pi -p "Summarize this codebase"
cat README.md | pi -p "Summarize this text"
pi -p @screenshot.png "What's in this image?"
```

---

## 3. 消息队列

| 操作 | 说明 |
|------|------|
| Enter | 排队转向消息（当前助手回合工具调用完成后发送） |
| Alt+Enter | 排队后续消息（助手全部工作完成后发送） |
| Escape | 终止并恢复排队消息到编辑器 |
| Alt+Up | 将排队消息召回编辑器 |

## 4. 快捷键汇总

### 全局导航

| 快捷键 | 说明 |
|--------|------|
| Ctrl+O | 打开会话（浏览/选择/创建） |
| Ctrl+Shift+O | 打开会话树视图 |
| Ctrl+S | 保存当前会话（手动提交） |
| Ctrl+L | 清除消息区域（保留上下文） |
| Ctrl+C | 中断当前操作 |

### 编辑器

| 快捷键 | 说明 |
|--------|------|
| Enter | 发送消息 |
| Shift+Enter | 换行 |
| Tab | 路径补全 |
| Ctrl+G | 打开外部编辑器 |
| Ctrl+A | 全选 |
| Ctrl+U | 清除当前行 |

### 会话导航

| 快捷键 | 说明 |
|--------|------|
| Ctrl+Shift+[ | 切换到父分支 |
| Ctrl+Shift+] | 切换到子分支 |
| Ctrl+Shift+\ | 切换到下一分支 |

### 斜杠命令

| 命令 | 说明 |
|------|------|
| `/login` | 登录提供商 |
| `/logout` | 登出 |
| `/session` | 查看/管理会话 |
| `/model` | 切换模型 |
| `/thinking` | 切换思考级别 |
| `/tools` | 管理工具开关 |
| `/compact` | 手动触发压缩 |
| `/clear` | 清除会话 |

## 5. 思考级别

| 级别 | 描述 | 适用场景 |
|------|------|---------|
| `off` | 无思考 | 简单任务、快速响应 |
| `minimal` | 最小 | 简单问答 |
| `low` | 低 | 简单编码任务 |
| `medium` | 中（默认） | 大多数任务 |
| `high` | 高 | 复杂架构设计 |
| `xhigh` | 极高 | 复杂问题解决 |
